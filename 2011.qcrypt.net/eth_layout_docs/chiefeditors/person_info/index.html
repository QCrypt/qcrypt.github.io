Persönliche Homepage

Navigieren Sie im Browserfenster zum Verzeichnis Personen (people) und wechseln Sie dann mittels Eingabe /edit in das SMI.

Wechseln Sie dann in das Verzeichnis, welcher die persönlichen Verzeichnisse enthält (Name z.B. Mitarbeiterverzeichnis, Köpfe o.ä.).

Wählen Sie jetzt im Aufklappmenü  select: "Silva Personal Information". Im nachfolgend angezeigten Formular geben Sie z.B. Namen und Vornamen ohne Umlaute ein (ü = ue usw.)


Treffen mehrere Angestellte auf diese Kriterien zu, so wählen Sie im nachfolgenden Fenster via Checkbox die gewünschte Person aus.

Trifft nur eine Person auf die Kriterien zu, so werden aus den Datenbanken die Daten geholt und zu einem Folder zusammengestellt.

Möchten Sie die persönliche Homepage mit einem Bild ergänzen, so legen Sie ein Bild Namens index mit Extension .jpg, .png oder .gif in das betreffende persönliche Verzeichnis. Es wird automatisch auf eine Standardgrösse skaliert.

Achtung: das index-Dokument dieses Folders muss noch publiziert werden.

Optionen

Im Kriterienfenster können folgende Optionen gewählt werden.

Mit der Angabe der default role kann den InhaberInnen einer Homepage automatisch Autoren oder Editorenberechtigungen zugewiesen werden.

Mit einem Häkchen für reversed naming wird die Sortierreihenfolge für Listen mittels Nachnamen gemacht. Wird das Häkchen entfernt, sortiert Silva nach dem Vornamen.
